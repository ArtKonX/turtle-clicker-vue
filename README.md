# Веб-приложение: кликер черепахи на vue

![CI](https://github.com/ArtKonX/turtle-clicker-vue/actions/workflows/web.yml/badge.svg)

[Ссылка на Github Pages](https://artkonx.github.io/turtle-clicker-vue/)

## Настройка и запуск

1. Клонирование репозитория
```
git clone https://github.com/ArtKonX/turtle-clicker-vue.git
```

2. Установка зависимостей
```
npm install
```

3. Запуск
```
npm run dev
```

## Стек:
1. Vue 3
2. SCSS
3. Vite
4. TypeScript
5. GitHub Pages

Я разработал игру‑кликер «Тортильник» с необычной игровой механикой и стилистикой. Хочу предложить проект для рассмотрения в рамках вашей игровой платформы.
Что представляет собой игра
Главный персонаж — черепаха с системой пассивного дохода.
Прогрессия — прокачка уровней, покупка улучшений, магазин скинов с отсылками к культовым фильмам.
Ключевая фишка — механика «Новая игра +»:
При достижении 9 999 уровня игрок может купить особый «X‑материал».
После покупки:
Весь прогресс сбрасывается (уровни, текущая валюта).
Сохраняются все купленные улучшения и скины.
Открывается секретная надпись с отсылкой к «Тортильник 2.0».
Добавляется секретный скин черепахи в виде летающей тарелки)
Почему это интересно
Вирусный потенциал — юмор, отсылки стимулируют делиться игрой.
Долгоиграющий геймплей — механика «Новая игра +» мотивирует возвращаться.
Техническая простота — проект на Vue 3, сохранение через localStorage.
